# AFO Ride Driver - Deployment Instructions

## Overview
This package contains the complete AFO Ride Driver application, including frontend (React + TypeScript) and backend (Motoko) source code, along with all necessary assets and configuration files.

## Prerequisites
- Node.js (v18 or higher)
- npm or pnpm package manager
- DFX SDK (for Internet Computer deployment)

## Package Contents
- `/frontend` - Complete React TypeScript frontend application
- `/backend` - Motoko backend canister code
- `/frontend/public/assets/generated` - Generated logo and assets
- Configuration files (package.json, tsconfig.json, tailwind.config.js, etc.)
